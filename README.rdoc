
This is a logbook application to record, track and display the results of workouts and bodycomposition changes.


Features:

- User login/account
- Personal profile and account settings
- Email confirmation
- Contact form

- Bodystats and calory entry/tracking page (weight, bodyfat, bodymeasurements)
- Workout entry/tracking page 
- One rep max calculator, maintenance/BRM calculator
- Results tracking/display via graphs.
